ğŸ® GAME FILES (Client)
----------------------
client/index.html
  - Entry point HTML
  - Game container styling
  - Loads main.js module

client/main.js
  - Phaser game configuration
  - Scene registration
  - Physics setup (Arcade)
  - Canvas: 1280x720

client/scenes/BootScene.js
  - Asset loading scene
  - Progress bar display
  - Placeholder graphics generation
  - Transitions to MenuScene

client/scenes/MenuScene.js
  - Main menu UI
  - Title and controls display
  - Play button with hover effects
  - Transitions to GameScene

client/scenes/GameScene.js
  - Core gameplay implementation
  - WASD movement (200 speed)
  - Mouse aiming & rotation
  - Shooting mechanics (600 bullet speed)
  - Procedural map (40x30 tiles)
  - Collision detection
  - Camera follow
  - UI (health, ammo)

client/vite.config.js
  - Build configuration
  - Output: dist/
  - Code splitting (Phaser separate)
  - Dev server: port 3000

client/package.json
  - Dependencies: Phaser 3.70.0
  - Dev dependencies: Vite 5.0.0
  - Scripts: dev, build, preview

.github/workflows/deploy.yml

  - Secrets: FIREBASE_SERVICE_ACCOUNT, FIREBASE_PROJECT_ID

TECHNOLOGY STACK
================

Frontend
--------
- Phaser 3.70.0      - HTML5 game engine
- Vite 5.0.0         - Build tool & dev server
- Vanilla JavaScript - ES6 modules

Backend
-------
- Firebase Hosting   - Static file hosting (CDN)
- Cloud Firestore    - NoSQL real-time database
- Cloud Functions    - Serverless backend (Node.js 18)
- Firebase Auth      - User authentication (ready)

DevOps
------
- GitHub             - Version control
- GitHub Actions     - CI/CD automation
- Replit             - Cloud IDE


GAME SPECIFICATIONS

Display
-------
- Resolution: 1280x720 pixels
- Rendering: WebGL (Canvas fallback)
- Target FPS: 60

Physics
-------
- Engine: Arcade Physics
- Gravity: Disabled (top-down)
- Collisions: AABB

Gameplay
--------
- Player Speed: 200 px/s
- Bullet Speed: 600 px/s
- Map Size: 40x30 tiles (32px each)

Controls
--------

- Mouse: Aiming
- Left Click: Shoot


DEPLOYMENT FLOW
===============

Development (Replit)
--------------------
1. Edit code in Replit
2. npm run dev
3. Test in webview
4. Commit changes

Automated Deploy (GitHub Actions)
----------------------------------
1. Push to main branch
2. GitHub Actions triggered
3. Install dependencies
4. Build client (Vite)
5. Deploy to Firebase
6. Live at: YOUR_PROJECT_ID.web.app

Manual Deploy (CLI)
-------------------
1. bash deploy.sh
2. Script builds & deploys
3. Live at: YOUR_PROJECT_ID.web.app
